# SecureOps

I want SecureOps to be a secure incident detection and audit logging platform that will be built with TypeScript, Next.js, Rust Axum, PostgreSQL and Docker.

The project will be focused on security operations, role-based access control, detection rules, audit trails and maintainable full-stack engineering.

## Planned Tech Stack

- Frontend: Next.js, React, TypeScript
- Backend: Rust, Axum
- Database: PostgreSQL
- Infrastructure: Docker Compose
- Planned: SQLx, authentication, role-based access control, audit logging, detection rules, CI and OpenAPI documentation

## Roadmap

- Incident model and API
- Security events model and API
- Detection rule engine
- Audit logging
- Authentication
- Role-based access control
- Frontend dashboard
- AI incident triage assistant
